---
title: 'Wishlist Empty State'
description: 'This element is a div next to the wishlist that is shown when the list is empty. It is used to inform the user that there are no products in the wishlist.'
---

<img src="/images/wishlist-empty.png" />
</Frame>

## Setup

<Steps>
 <Step title="Wishlist Empty State Wrapper">
    Add a empty div block as sibling to the  wishlist.
 </Step>
 <Step title="Wishlist Empty State Wrapper - Attributes">
   Add following attribute to the div block:
    ```html
    looop=wishlist_empty-state
    ```
 </Step>
</Steps>
