---
title: 'Orders Empty State'
description: 'This element is a div next to the orders list that is shown when the list is empty. It is used to inform the user that there are no orders to display.'
---

<Frame>
<img src="/images/orders-empty.png" />
</Frame>

## Setup

<Steps>
 <Step title="Orders Empty State Wrapper">
    Add a empty div block as sibling to the orders list.
 </Step>
 <Step title="Orders Empty State Wrapper - Attributes">
   Add following attribute to the div block:
    ```html
    looop=orders_empty-state
    ```
 </Step>
</Steps>
