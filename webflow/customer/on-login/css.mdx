---
title: 'CSS Variables'
description: 'These sub-variables are used to change the CSS properties of the element based on the user login status.'
---

## Avilable Sub-Variables

### Hide
```html
looop_on-login=hide
```
### Show
```html
looop_on-login=show
```

### Flex
```html
looop_on-login=flex
```

### Disable
```html
looop_on-login=disable
```

### Enable
```html
looop_on-login=enable
```
