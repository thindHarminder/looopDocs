title: 'Reset'
description: 'Allow users to reset their password from the email they receive.'
---

## Setup
<Steps>
<Step title="Reset Form">
Add a Webflow form to the Reset page.
</Step>
<Step title="Reset Form - Attribute">
 Add the following attribute to the `Form` element of the Webflow form:
    ```html
    looop=resetform
    ```
</Step>
<Step title="Reset Form - Fields">
Add [available fields](#available-fields) to the form
</Step>
<Step title="Success Redirect">
In the `Form` element settings, set the `Redirect URL`. On successful signup, the user will be redirected to this URL.
<Frame>
<img src="/images/form-redirect.png" />
</Frame>
</Step>
</Steps>

## Available Fields 

### New Password <sup>`required`</sup>
```html
looop=new_password
```

### Confirm New Password <sup>`required`</sup>
```html
looop=confirm_new_password
```
