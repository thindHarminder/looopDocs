---
title: 'CSS Variables'
description: 'These sub-variables are used to change the CSS properties of the element if the user is logged out.'
---

## Avilable Sub-Variables

### Hide
```html
looop_on-logout=hide
```
### Show
```html
looop_on-logout=show
```

### Flex
```html
looop_on-logout=flex
```

### Disable
```html
looop_on-logout=disable
```

### Enable
```html
looop_on-logout=enable
```
