---
title: 'Cart List'
description: 'The cart list is a collection list of all the variants you have in the Variants CMS collection.'
---

The cart items are dynamically shown and hidden from the user depending on what they have in their cart.

## Setup

1. Add a new collection list inside the [default wrapper](/webflow/cart/default/wrapper) element.
2. Add the custom attribute to the webflow collection list item element.
<Frame>
<img src="/images/cart-list-item.jpg" />
</Frame>
```html
looopCartItem=SET_DYNAMIC_VALUE_OF_VARIANTS_LOOOP_ID
```
3. Develop the [cart item ](/webflow/cart/default/cart-item)by adding quantity input, remove button, and the variant image.
