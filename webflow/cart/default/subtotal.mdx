---
title: 'Subtotal'
description: 'Subtotal can be any text element inside the default cart wrapper. This element will show the subtotal of all items in the cart after applicable discounts.'
---

## Setup

Add the following code to your cart template where you want the subtotal to appear:

```html
"looop"="subtotal"
```
