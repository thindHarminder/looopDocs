---
title: 'Go to Cart'
description: 'The button will recirect the user to the cart page of your Shopify site'
---

Many apps run on the Shopify cart page and if you want to use them, you need to make sure that the user is redirected to the cart page before checkout. This is the button that will redirect the user to the cart page of your Shopify site.

```html
"looop"="shopifycart"
```
