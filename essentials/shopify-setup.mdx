## Prerequisites
1. Active Shopify account
2. User with Admin access to the Shopify account

## Step 1: Create a Private App in Shopify
1. Log in to your Shopify account.
2. From your Shopify admin, go to `Apps` > `App and sales channel settings`.
3. Click on `Develop apps`.
<Note>
If you have not developed any custom apps before for the selected store, Shopify will ask you to enable permissions to build custom/private apps. 
</Note>
4. Click on `Create an app`.
5. Enter the app name and click on `Create app`.   

## Step 2: Configure API scopes
1. Click the `Configure Admin API scopes` button.
2. Select the following scopes:
   <Adminscopes />
3. Click `Save`.
4. Go to `Configuration` Tab
5. Click `Configure` next to Storefront API
6. Select all the available scopes and click `Save`.

## Step 3: Install the app in your store
1. Click on `Install app` in the top right corner.
2. Click on `Install` again in the confirmation dialog.
3. Once the app is installed, you will be redirected to the app credentials page.
<Tip>
You will see two API key - Admin API access token and Storefront API access token. Dont Reveal the Admin API access token yet as it will be visible only once.
</Tip>

## Step 4: Get the Shopify store URL
1. Go to your Shopify admin.
2. Go to `Settings` > `Domains`.

## Step 5: Connect Shopify to Looop
1. Now copy reveal the Admin API access token and Storefront API access token from the Shopify app credentials page.
2. On Looop Shopify setup page enter the following details:
   - Shopify Store URL [Step 4](#step-4-get-the-shopify-store-url)
   - Admin API access token genrated in [Step 3](#step-3-install-the-app-in-your-store)
   - Storefront API access token genrated in [Step 3](#step-3-install-the-app-in-your-store)
3. Click `Save & Connect`.
4. Looop will verify the credentials and connect to your Shopify store.
<Note>
If you see an error, please verify the credentials and try again.
</Note>
