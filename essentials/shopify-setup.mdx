## Prerequisites
1. Active Shopify account
2. User with access to install Shopify apps

## Install Looop Shopify App
1. Go to [Looop Shopify App](https://apps.shopify.com/looop-build-with-webflow) and click on `Install` button.

<Note>
If you see an error, please verify the credentials and try again.
</Note>
